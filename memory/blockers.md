## Blocker Categories

Use these tags in titles:
- `[ENV]` - Environment/infrastructure
- `[DEP]` - Dependency issue
- `[BUG]` - Bug in target code
- `[DESIGN]` - Needs design decision
- `[EXTERNAL]` - External system/API
- `[UNCLEAR]` - Requirements unclear

## Escalation Protocol

1. **LOW**: Log and continue, mention in daily summary
2. **MEDIUM**: Director decides if to pause or work around
3. **HIGH**: Director pauses related work, attempts resolution
4. **CRITICAL**: Full stop, create alert, wait for human
